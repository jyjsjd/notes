# Here Document

```
cmd<<[−]delimiter 
    here-document
delimiter
```

它的作用就是将两个 `delimiter` 之间的内容(Here Document Content 部分) 传递给cmd 作为输入参数。

‘<<-’, then all leading tab characters are stripped
